Maker. I build small, sharp tools for Linux and the web, and keep them minimal
on purpose — plain shell over daemons, flat SVGs over icon packs, no dependency
I can't read in one sitting.

**Out in the open**

- [**xfce-ufw-indicator**](https://github.com/anton-amat/xfce-ufw-indicator) — a tiny firewall shield for the Xfce panel. One script, two SVGs, no daemon.
- [**xfce-panel-mono**](https://github.com/anton-amat/xfce-panel-mono) — monochrome CPU / RAM / temp / network / volume monitors for the same panel.

If something here saves you five minutes, it did its job. Issues and PRs welcome.
